# bCat_pPAX

* **Developed for:** Laurent
* **Team:** Germain
* **Date:** October 2023
* **Software:** Fiji

### Images description

2D images of adherens junctions.

2 channels:
  1. bCat
  2. pPAX

### Macro description

* Detect bCat signal in corresponding channel with Gaussian blur filtering + thresholding + median filtering
* Detect pPAX signal in corresponding channel with median filtering + thresholding + median filtering
* Give bCat area and pPAX area in/out of bCat signal
